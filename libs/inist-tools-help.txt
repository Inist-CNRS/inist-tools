Commandes :

  inist <service> [help|on|off|status] 
  
  on | off    Positionne ou supprime le proxy INIST pour le service spécifié.
  help        Informe des éventuelles particularités pour <service>
  status      Indique l'état actuel du paramétrage du proxy pour le <service>
  
  Par défaut, la commande inist <service> sans argument donne le statut actuel
  du proxy pour le service concerné.
  
  <services> disponibles :
  
    * apt     Paramètre le proxy inist pour apt
    * bower   Paramètre le proxy inist pour bower
    * docker  Paramètre le proxy inist pour docker
    * env     Paramètre le proxy inist pour l'environnement et les shells
    * gnome   Paramètre le proxy inist pour gnome
    * kde     Paramètre le proxy inist pour kde
    * npm     Paramètre le proxy inist pour npm
    * ntp     Synchronise l'horloge de votre PC avec le serveur de temps de
              l'INIST
    * shell   (alias de env)
    * unity   Paramètre le proxy inist pour unity
    * xfce    Paramètre le proxy inist pour xfce

    * curl    Paramètre le proxy inist pour curl. Curl peut aussi fonctionner en
              utilisant les variables d'environnement positionnée avec 
              <inist env on>.
    * wget    Paramètre le proxy inist pour wget. Wget peut aussi fonctionner en
              utilisant les variables d'environnement positionnée avec 
              <inist env on>.

Options :

  -h, --help      Cette aide

  -i, --info      Donne des informations sur votre configuration

  -r, --reload    Recharge inistrc pour prendre en compte les dernières
                  modifications apportées à inistrc (orienté dev)

  -s, --status    Renseigne sur l'état actuel du paramétrage du proxy INIST
                  pour l'environnement système
            
  -v,--version    Affiche la version
