Commandes :

  info      Donne des informations sur votre configuration

  proxy     [on|off] [--chrome|--chromium|--firefox|--iceweasel|
                      --kde|--unity|--gnome|--xfce|
                      --apt|--npm]
            Positionne ou supprime le proxy INIST pour la session en cours ou
            pour le navigateur
            Utilisée seule, cette commande positionne les variables
            d'environnement pour utiliser le proxy INIST dans la session
            courrante (shell).
            on            : positionne les variables d'environnement pour le
                            proxy INIST 
            off           : supprime les variables d'environnement pour le
                            proxy INIST 
            on  --<appli> : configure <appli> pour utiliser le proxy INIST
            off --<appli> : retire le proxy INIST de la configuration de <appli>

  reload    recharge inistrc pour prendre en compte les dernières modifications 
            apportées à inistrc (orienté dev)

  sync      synchronise la date et l'heure de la machine sur le serveur de temps
            de l'INIST
  
  time      alias de sync
  date      alias de sync
            
Options :
  -v,--version    Affiche la version
